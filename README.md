## Getting Started

### Prerequisites

To run this project, you'll only need to have the npm installed.

### Installing

Cloning the Repository

```
$ git clone https://github.com/rodrilima/storecash.git

$ cd storecash

$ npm install
```

### Running

```
$ npm start
```
